## 如何使用

要运行此应用程序，请先 fork 此存储库。

在您 fork 的仓库中，转到“代码”下的“代码空间”选项卡。
点击“在主线程上创建代码空间”将开始创建环境。
请稍等片刻，待 Rails 服务器启动并显示 Simple Browser 后即可使用。

接下来，运行数据库迁移。

```
$ rails db:migrate
```

最后，运行测试以确保其正常工作。

```
$ rails 测试
```
